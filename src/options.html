Prompt in a Box

Your prompt is the program. This page configures the runtime and audits every action the agent takes.

First-time setup

  1. Pick a model provider below.
  2. Paste its API key. Keys are stored only in this browser, and sent only to the provider's endpoint.
  3. Click Test to prove the key works, then Save & finish.
Provider
Any model ID the SDK accepts. Default is tracked in config.ts.
Provider tools
Server-executed tools the provider runs on your behalf — web search, web fetch, code execution. They count against the provider's API usage like any other call. Turn off any you don't want the agent to use.
Schedule
Minutes between automatic runs. The service worker dies between invocations; this is the only thing that wakes it.
